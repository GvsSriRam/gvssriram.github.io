Work

Balance-AI

Balance AI features an LLM chatbot that delivers insights in a conversational manner, robust predictive models for forecasting, optimization, and tailored recommendations, and interactive dashboards and reports for visualizing key trends and insights.

Talk-to-the-Hand-Sign-Language-Recognition

A sign language recognition system that leverages computer vision to translate hand gestures into text, built using Python, TensorFlow, and FastAPI for the backend, and HTML, CSS, and JavaScript for the frontend, empowers deaf and hard-of-hearing individuals to communicate more effectively. Hosted on GCP, this project demonstrates my expertise in building scalable and reliable cloud-based applications.

Flight-Arrival-Delay-Prediction

A flight arrival delay prediction system, built as a classroom project for Intro to ML, where I achieved first place, uses historical flight arrival and departure times, weather data, and various tree-based classification models to predict delays for the upcoming five days. The system also analyzes the impact of one delay on subsequent flights, demonstrating my expertise in machine learning and data analysis.

CRED-:-Credit-Card-Management-Chatbot

A chatbot for managing credit card accounts, uses NLP to understand user intents and provide relevant responses, showcasing my expertise in NLP and chatbot development.

Whisper Wallet

AI-enhanced expense recognition system, uses CV and NLP to accurately parse expenses from scanned or photographed bills, simplifying expense tracking and sharing.
